# 프로젝트 상태 확인 (Dev Status)

현재 프로젝트의 전체 상태를 한눈에 파악합니다.

## 수행 작업

다음 항목들을 순서대로 확인하고 **요약 리포트**를 출력하세요.

### 1. 프로젝트 정보
- `mix.exs`에서 `@version` 읽기
- 앱 이름, Elixir 버전 확인

### 2. Git 상태
- `git status` - 변경/추가/삭제된 파일
- `git log --oneline -10` - 최근 10개 커밋
- 현재 브랜치

### 3. 코드베이스 규모
- `lib/` 하위 `.ex` 파일 개수
- `test/` 하위 테스트 파일 개수
- 주요 모듈 목록

### 4. 의존성 상태
- `mix.exs`의 deps 목록 요약

### 5. 문서 상태
- `guides/` 가이드 파일 목록
- ex_doc 문서 존재 여부 (`doc/` 디렉토리)

## 출력 형식

```
========================================
  프로젝트 상태 리포트
========================================

버전: v0.X.X
브랜치: main
마지막 커밋: <hash> <message>

코드: XX개 모듈 | XX개 테스트
변경: XX개 파일 수정됨

최근 커밋 (5개):
  - <hash> <message>
  - ...

가이드: XX개 (한국어 X + 영어 X)
========================================
```
