#### 2. 데이터구조명세서.md
**목적:** Grid 내부 데이터 구조 및 assigns 정의  
**내용:**
- LiveView assigns 스키마
- 컬럼 정의 구조 (Column Schema)
- 필터/정렬 상태 구조
- 페이지네이션 상태

**예시:**
```elixir
%{
  data: [%{id: 1, name: "Alice"}],
  columns: [%{field: :name, label: "이름"}],
  sort: %{field: :name, direction: :asc},
  filters: %{name: "Ali"}
}
```
