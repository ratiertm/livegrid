```
Phase 1 (v0.8) - 핵심 UX 완성
  F-901  조건부 셀 스타일      ← 넥사크로 핵심 기능

  F-511  Excel Import          ← Export 짝
  F-900  셀 툴팁              ← 기본 UX

Phase 2 (v0.9) - 편집 고도화
  F-905  Checkbox 컬럼
  F-920  행 단위 편집
  F-922  입력 제한
  F-700  Undo/Redo
  F-935  정렬 null 처리

Phase 3 (v1.0) - 엔터프라이즈
  F-940  셀 범위 선택
  F-941  선택 영역 합계
  F-800  컨텍스트 메뉴
  F-903  Suppress
  F-500  실시간 협업
```
