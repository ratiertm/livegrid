# 백엔드 스킬 목차 (컨텍스트 / Ecto / 비즈니스 로직)

> 이 목차를 먼저 읽고, 현재 작업에 해당하는 챕터만 펴보세요.

## 챕터 목록

| 챕터 | 파일 | 언제 읽나 |
|------|------|-----------|
| Context 설계 | `context-design.md` | 컨텍스트 모듈 생성/수정 시 |
| Ecto 패턴 | `ecto-patterns.md` | 스키마, 체인지셋, 쿼리, 마이그레이션 작업 시 |

## 전체 적용 규칙 (항상)
- 비즈니스 로직은 `lib/content_flow/` 하위 Context 모듈에 작성
- 웹 레이어(`content_flow_web`)에서 `Repo` 직접 호출 절대 금지
- 모든 공개 함수에 `@doc`과 `@spec` 작성
- 패턴 매칭 우선 — `if/else` 중첩보다 `case`, `with`, 함수 클로즈 다중 정의 활용
- 파이프 연산자(`|>`) 적극 사용 — 가독성 우선
