## 주요 발견사항

1. **REST API 인증 이슈 (F-400)**: /api-demo 페이지에서 그리드가 0건 표시. REST API가 Bearer 토큰 인증 필요하나 DataSource.Rest에 API 키가 설정되지 않음
2. **Advanced Demo 범위**: /advanced-demo는 v0.7 기능(그룹핑, 트리, 피벗, 포맷터)만 탭으로 제공. v0.10 넥사크로 기능(Suppress, 셀병합, 줄바꿈, 행고정 등)은 별도 데모 탭 없음
3. **컬럼 필터 UI**: 컬럼 헤더의 필터 드롭다운이 클릭으로 열리지 않아 F-060/061/062 테스트 불가
4. **Export/Import**: CSV 다운로드(F-045), Excel/CSV Export(F-510), Excel Import(F-511) 버튼이 UI에서 미발견
5. **Grid Config Demo**: /grid-config-demo 페이지 직접 접근 불가했으나, /demo의 ⚙ 설정 버튼으로 동일 모달 테스트 완료
