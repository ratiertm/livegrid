# Master / Detail

행을 클릭하면 하위 디테일 영역이 펼쳐지는 마스터-디테일 UI입니다.

## Overview

Master/Detail은 요약 행(Master)을 클릭하면 상세 정보(Detail)가 아래에 펼쳐지는 패턴입니다. 주문-상품, 부서-직원 같은 1:N 관계 데이터에 적합합니다.

## Toggle UI

각 마스터 행에 ▶/▼ 토글 아이콘이 표시됩니다:

```
▶ 주문 #001  Alice  2026-03-01  ₩50,000
▼ 주문 #002  Bob    2026-03-02  ₩30,000
  ├─ 상품A  ₩15,000  x2
  └─ 상품B  ₩10,000  x1 (접혀 없어질 수 있음)
▶ 주문 #003  Carol  2026-03-03  ₩80,000
```

- **▶** (접힘): 클릭하면 디테일 펼침
- **▼** (펼침): 클릭하면 디테일 접기

## Behavior

- 토글 아이콘 클릭으로 개별 행의 디테일을 펼치고 접습니다
- 여러 행을 동시에 펼칠 수 있습니다
- 정렬/필터 시 펼침 상태가 유지됩니다
- 페이지 이동 시 펼침 상태는 초기화됩니다

## CSS Classes

| 클래스 | 적용 대상 |
|--------|----------|
| `.lv-grid__row--master` | 마스터 행 |
| `.lv-grid__row--detail` | 디테일 행 |
| `.lv-grid__toggle` | 토글 아이콘 (▶/▼) |
| `.lv-grid__toggle--expanded` | 펼쳐진 상태 아이콘 |
