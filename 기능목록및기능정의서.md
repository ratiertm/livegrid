## 버전별 기능 로드맵

### 전체 로드맵

```
v0.1 (핵심 그리드)              ✅ 완료
  ↓
v0.2 (필터/검색/검증/테마)       ✅ 완료
  ↓
v0.3 (대용량 처리 + DBMS 연동)   ✅ 완료
  ↓
v0.4 (셀 편집 + 컬럼 조작)       ✅ 완료
  ↓
v0.5 (고급 기능 + REST API)      ✅ 완료
  ↓
v0.6 (DBMS & API 강화)          ⚠️ Phase A 완료, B~D 미완
  ↓
v0.7 (고급 데이터 처리)          ✅ 완료 ← 현재
  ↓
v0.8 (협업 & 실시간)
  ↓
v1.0 (엔터프라이즈)
```

### 티어별 기능 분류

| 티어 | 버전 | 주요 기능 | 목표 사용자 |
|------|------|----------|-------------|
| **Community** | v0.1~v0.3 | 기본 CRUD, 정렬, 필터, 페이징, 가상 스크롤, DBMS 연동 | 개인/오픈소스 |
| **Professional** | v0.4~v0.7 | 편집, 커스텀 렌더러, REST API, 그룹핑, 피벗, 트리 | 스타트업/중소기업 |
| **Enterprise** | v0.8~v1.0+ | 실시간 협업, 대용량 최적화, 멀티 DB, 키보드 내비게이션 | 대기업 |
